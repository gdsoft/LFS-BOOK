6.33. Grep-3.3

Grep 软件包包含用于在文件中搜索的程序。

大致构建用时: 0.4 SBU
所需磁盘空间: 37 MB

6.33.1. 安装 Grep

准备编译 Grep ： 

./configure --prefix=/usr --bindir=/bin

编译软件包： 

make

用以下命令测试结果： 

make -k check

安装软件包： 

make install

6.33.2. Grep 软件包内容

安装的程序: egrep, fgrep, 和 grep

简要介绍

egrep

打印匹配扩展正则表达式的行

fgrep

打印匹配固定字符串列表的行

grep

打印匹配基本正则表达式的行
