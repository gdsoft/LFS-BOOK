6.38. Expat-2.2.6

Expat 软件包包含一个用于解析 XML 的面向流的 C 库。

大致构建用时: 0.1 SBU
所需磁盘空间: 11 MB

6.38.1. 安装 Expat

首先修复一个 LFS 环境中，回归测试的问题： 

sed -i 's|usr/bin/env |bin/|' run.sh.in

准备编译 Expat： 

./configure --prefix=/usr    \
            --disable-static \
            --docdir=/usr/share/doc/expat-2.2.6

编译软件包： 

make

用以下命令测试结果： 

make check

安装软件包： 

make install

如果需要的话，安装帮助文档： 

install -v -m644 doc/*.{html,png,css} /usr/share/doc/expat-2.2.6

6.38.2. Expat 软件包内容

安装的程序: xmlwf
安装的库: libexpat.so
安装目录: /usr/share/doc/expat-2.2.6

简要介绍

xmlwf

用于检查 XML 文档是否格式良好的非验证工具

libexpat

包含用于解析 XML 的 API 函数
